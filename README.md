# 🎮 Gaming Network Optimizer

윈도우 11 환경에서 이더넷 및 와이파이 네트워크의 응답 속도(Latency)를 최적화하여 온라인 게임의 핑(Ping)을 안정화해주는 오픈소스 도구입니다.

## 🚀 주요 기능

- **TcpAckFrequency 설정**: 패킷 응답 확인 빈도를 높여 지연 시간을 단축합니다.
- **TCPNoDelay (Nagle)**: 작은 데이터 패킷을 즉시 전송하여 반응 속도를 개선합니다.
- **NetworkThrottlingIndex 해제**: 멀티미디어 재생 시 발생하는 네트워크 제한을 제거합니다.
- **일괄 적용 지원**: 단 한 번의 클릭으로 PC의 모든 네트워크 어댑터에 설정을 적용합니다.
- **자동 관리자 권한 요청**: 레지스트리 수정을 위해 실행 시 관리자 권한을 자동으로 호출합니다.
